Trabalando com Cores Gradientes

Os gradientes CSS permitem exibir transições suaves entre duas ou mais cores especificadas.

CSS define dois tipos de gradientes:

  • Gradientes Lineares (desce / sobe / esquerda / direita / diagonalmente)
  • Gradientes radiais (definidos pelo centro)

Gradientes Lineares (Desce e Sobe)

Para criar um gradiente linear, você deve definir pelo menos duas cores. Podemos também definir um ponto de partida e uma direção (ou um ângulo) junto com o efeito de gradiente.
